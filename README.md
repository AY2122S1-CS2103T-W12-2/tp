# SWEe-book
**SWEe-book** is a desktop app that makes it easier for students to 
**keep track of their tasks and group mates in different groups**. 

### Features
1. Adding a task	
2. Deleting a task
3. Retrieving contact information of a group 
4. Sorting tasks
5. Filtering tasks by modules
6. Listing all tasks
